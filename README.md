Sistema interno de gerenciamento para salão de beleza, desenvolvido com Angular.

Projeto criado para controle de:
- agenda de profissionais
- agendamentos
- painel administrativo
- controle financeiro
- comissões
- autenticação
- gerenciamento operacional do salão

---

# ✨ Funcionalidades

## 👩‍💼 Profissionais
- Login individual para cada profissional
- Controle de privacidade
- Agenda personalizada
- Visualização apenas dos próprios agendamentos

---

## 📅 Agenda Inteligente
- Agenda visual por horário
- Organização diária
- Funcionamento de terça a sábado
- Horários das 08h às 18h
- Calendário visual para próximos 365 dias

---

## 💰 Painel Administrativo
- Controle financeiro
- Valor dos serviços
- Cálculo de comissão
- Gestão dos profissionais
- Gestão dos agendamentos

---

## 📱 Responsividade
- Desktop
- Tablet
- Mobile
